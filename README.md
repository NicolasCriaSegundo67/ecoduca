# EcoEduca 🌱

Projeto desenvolvido com foco na ODS 4 – Educação de Qualidade.

## Objetivo
Promover conscientização ambiental através da tecnologia.

## Tecnologias
- HTML
- CSS
- JavaScript

## Funcionalidades
- Quiz sustentável
- Dicas ecológicas
- Calculadora de economia de água
- Interface moderna

## Como executar
1. Baixe os arquivos.
2. Abra o `index.html` no navegador.
